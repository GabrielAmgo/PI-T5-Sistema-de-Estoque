# PI-T5-Sistema-de-Estoque
## Projeto Integrador Univesp - Turma 5

### Desenvolvimento de uma aplicação desktop em Java com persistência em MySQL para o controle de estoque em pequenos comércios.

### Funcionalidades Escopadas:
 - Controle de Ativos: Cadastro, edição e exclusão de produtos com monitoramento de ativos.
 - Gestão de Fluxo: Registro sistemático de entradas e saídas para integridade de dados.
 - Log de Movimentação: Histórico de transações para auditoria básica de estoque.
